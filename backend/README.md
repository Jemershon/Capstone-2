# Notetify Backend

MERN backend for Google Classroom-like system.

## Setup

1. Make sure MongoDB is running locally.
2. Install dependencies:
   ```
   npm install
   ```
3. Start server:
   ```
   npm start
   ```
4. API endpoints:
   - POST `/api/register`
   - POST `/api/login`
   - GET `/api/classes`
   - POST `/api/classes`
