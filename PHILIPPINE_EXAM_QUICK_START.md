## üöÄ Quick Start (5 Steps)

### Step 1: Create Form
Go to **Teacher Dashboard** ‚Üí **Forms & Surveys** ‚Üí **Create New Form**

### Step 2: Enable Philippine Style
- Click **Settings** (top right)
- Go to **üáµüá≠ Philippine Exam Style** tab
- Toggle **ON** "Use Philippine Exam Format"

### Step 3: Fill Exam Header
```

Teacher: Prof. Juan Dela Cruz  
Semester: First Semester
School Year: 2024-2025
Exam Date: November 15, 2024
Duration: 90 minutes
Passing Score: 60%
```

### Step 4: Create Sections
Click **"Add Section"** for each part:

**Part I:**
```
Title: Part I: Multiple Choice
Instructions: Choose the letter of the correct answer.
Points Per Item: 2
```

**Part II:**
```
Title: Part II: Identification
Instructions: Identify what is being described.
Points Per Item: 3
```

**Part III:**
```
Title: Part III: True or False
Instructions: Write T if true, F if false.
Points Per Item: 1
```

**Part IV:**
```
Title: Part IV: Enumeration
Instructions: Enumerate what is being asked.
Points Per Item: 5
```

**Part V:**
```
Title: Part V: Matching Type
Instructions: Match Column A with Column B.
Points Per Item: 2
```

### Step 5: Add Questions
For each section:
1. Click **"Add Question"**
2. Select **type** (True/False, Identification, etc.)
3. Choose **section** from dropdown
4. Enter **question text**
5. Set **correct answer** (for auto-grading)
6. Set **points** (or use section default)
7. Click **"Save Question"**

## üéØ Common Exam Formats

### Quick Quiz (15 min)
```
Part I: True or False (10 items x 1 pt = 10 pts)
Part II: Identification (5 items x 2 pts = 10 pts)
Total: 20 points
```

### Short Exam (30 min)
```
Part I: Multiple Choice (15 items x 2 pts = 30 pts)
Part II: True or False (10 items x 1 pt = 10 pts)
Part III: Identification (10 items x 1 pt = 10 pts)
Total: 50 points
```

### Midterm (90 min)
```
Part I: Multiple Choice (25 items x 2 pts = 50 pts)
Part II: True or False (10 items x 1 pt = 10 pts)
Part III: Identification (10 items x 2 pts = 20 pts)
Part IV: Enumeration (2 items x 5 pts = 10 pts)
Part V: Essay (2 items x 5 pts = 10 pts)
Total: 100 points
```

### Final Exam (120 min)
```
Part I: Multiple Choice (40 items x 2 pts = 80 pts)
Part II: True or False (20 items x 1 pt = 20 pts)
Part III: Matching Type (10 items x 2 pts = 20 pts)
Part IV: Identification (20 items x 2 pts = 40 pts)
Part V: Enumeration (4 items x 5 pts = 20 pts)
Part VI: Essay (4 items x 5 pts = 20 pts)
Total: 200 points
```

## ‚ö° Tips & Tricks

### Auto-Grading
- ‚úÖ Works for: Multiple Choice, True/False, Identification, Checkboxes, Enumeration, Matching
- üìù Manual needed for: Essays, Paragraphs, File Uploads

- Difficult questions: 5-10 points

### Section Organization
1. Start with objective (MC, TF)
2. Middle difficulty (ID, Enumeration)
3. End with subjective (Essay)

### Passing Score
- Quizzes: 70-75%
- Exams: 50-60%
- Finals: 50-55%
